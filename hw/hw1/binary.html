Binary For Dummies
General Counting Addition Conversion



In the world of computing, a binary number is a number based on a binary number system

which uses only two symbols usually 0 and 1. This number system is also known as a

base-2 numeral system which is similar but not exactly equivalent to the usual base-10

decimal number system we are used to working with.


Binary was implemented in the world of computing and electronics because the transistors

used in electronics only have two states, on and off, which makes it easy to convert those

states to 1's and 0's.
