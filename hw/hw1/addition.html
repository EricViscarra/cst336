Binary For Dummies
General Counting Addition Conversion



Addition is the simplest arithmetic operation to do in binary, you just have to remember the rules.

Lets start by adding two single digit binary numbers to explore these rules.

Follow along with the image on the left. To start we will be ignoring the top carry bit.

If we add two 0's we get 0, if we add 0 and 1 in any order then we get 1. But, if we

add 1 and 1, then we get 10, a 0 for this digit place's value, and a 1 to carry into the

next arithmetic operation. Now try it for yourself below!
