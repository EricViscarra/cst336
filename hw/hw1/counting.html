Binary For Dummies

General Counting Addition Conversion


Counting in binary is akin to counting in any other number system. Starting with just

one digit, counting proceeds with each symbol in increasing order. When a binary

number is incremented, the furthest right digit is changed from a 0 to a 1, but if that

digit is already 1 then it resets to 0 but causes the next digit to its left to then be

incremented.


As binary is a base-two number system, each additional digit serves as a

growing power of 2, where the rightmost digit represents 2^0 and every digit going left

of that initial number being a power of two greater. For Example: 2^1, 2^2, 2^3
